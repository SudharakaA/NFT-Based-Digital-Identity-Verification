# NFT-Based Digital Identity & Verification System 🚀

> A decentralized identity system powered by NFTs, ensuring trust, security, and ownership in the digital age.

![image](https://github.com/user-attachments/assets/e9fc954d-7012-40a5-bc68-b81fb9c246b7)


## 🌟 Overview

In an era where digital identity security is more critical than ever, we are building an NFT-powered decentralized identity system that ensures trust, security, and ownership like never before!

## 🔥 Key Features

- **Minting Digital Identity**: Create unique, tamper-proof identity as an NFT
- **On-Chain Verification**: Secure, decentralized authentication of identities
- **Immutable & Secure**: Leveraging Ethereum/Polygon blockchain
- **Decentralized Storage**: IPFS integration for enhanced security
- **Seamless UX**: Integration with MetaMask wallet

## 🌍 Use Cases

- 🔐 KYC & Identity Verification
- 🔑 Secure Login & Authentication
- ⭐ Decentralized Reputation Systems
- 📜 Digital Certificates & Credentials

## 🛠 Tech Stack

- Solidity - Smart Contracts
- Web3.js - Blockchain Interaction
- IPFS - Decentralized Storage
- React - Frontend Framework
- Node.js - Backend Services

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/SudharakaA/NFT-Based-Digital-Identity-Verification.git

# Install dependencies
cd NFT-Based-Digital-Identity-Verification
npm install

# Run the development server
npm run dev
```

## 📝 Environment Variables

Create a `.env` file in the root directory:

```env
PRIVATE_KEY=your_private_key
POLYGON_RPC_URL=your_polygon_rpc
IPFS_PROJECT_ID=your_ipfs_project_id
IPFS_PROJECT_SECRET=your_ipfs_secret
```

## 🔧 Smart Contract Deployment

```bash
npx hardhat compile
npx hardhat deploy --network polygon
```
